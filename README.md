!! Randoming Pass — генератор паролей

Мобильное приложение на .NET MAUI (C#, XAML).
УП ТЭПК группа 1942с Сеченов и Смокотин.

 !! Что внутри

- одна страница MainPage без БД и навигации;
- криптостойкая генерация через RandomNumberGenerator;
- цветная подсветка символов в пароле (заглавные строчные цифры спецсимволы);
- индикатор надёжности в виде одной полосы;
- ползунок длины от 4 до 64 символов;
- чекбоксы по типам символов и опция "без похожих" (I l 1 O 0);
- история последних 5 паролей в памяти при перезапуске пусто;
- кнопки "Копировать" и "Заменить" рядом с паролем;
- стандартная тёмная тема, серый акцент.

  !! Логика генерации

1. Из выбранных кнопок собирается общий пул символов.
2. Если включена опция "без похожих", из пула убираются I l 1 O 0 o.
3. Из пула криптослучайно набирается строка нужной длины.
4. Для каждого выбранного типа гарантированно подставляется хотя бы один символ в случайную позицию.
5. Надёжность считается по длине и наличию разных типов символов.
